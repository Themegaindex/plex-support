# Plex Support

Support- und Hilfeseite für den PrimeTime Plex-Streaming-Service.

**Live:** https://Themegaindex.github.io/plex-support

## Seiten

- **/** — Startseite
- **/hilfe** — Fehlerbehebung & Einrichtung
- **/faq** — Häufige Fragen
- **/dienste** — Serviceangebot
- **/kontakt** — Kontaktformular

## Entwicklung

```bash
npm install
npm run dev      # localhost:4321
npm run build    # Produktions-Build
```

## Deployment

Automatisch via GitHub Actions auf GitHub Pages bei jedem Push auf `main`.
